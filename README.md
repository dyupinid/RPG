**Быстрый запуск в браузере (с GitHub Codespaces):**

1) Откройте репозиторий на GitHub.
2) Нажмите зелёную кнопку "Code" в верхней части страницы.
3) Выберите "Open with Codespaces" → "Create codespace on main".
4) Codespace автоматически загрузится и запустит игру в терминале (без дополнительных команд).

**Описание**
Проект состоит из двух основных файлов:

RPG.py: Определяет основные механики боя, включая классы для Weapon (Оружие), Character (Персонаж), Player (Игрок) и Villain (Злодей). Обеспечивает атаки, расчёт урона, набор опыта, повышение уровня и многое другое.
text_adventure.py: Реализует консольную приключенческую игру, где игрок перемещается по комнатам, сражается с врагами с использованием боевой системы из RPG.py, собирает лут и обнаруживает секреты.

Игра полностью текстовая и запускается в терминале. Не требует внешних зависимостей, кроме стандартной библиотеки Python.

**Особенности**

Боевая система:
Пошаговые сражения с выбором действий для игрока (атака, лечение, щит, смена оружия).
Случайные элементы: критические удары, уклонения, комбо, контратаки.
Элементальные взаимодействия (огонь, вода и т.д.) для модификаторов урона.
Статусные эффекты (например, горение с уроном со временем).
Набор опыта, повышение уровня с улучшением характеристик (сила, ловкость, фокус).

Игровой процесс:
Навигация по комнатам с описаниями и выходами.
Встречи с врагами (одиночными или группами) с использованием боевой системы.
Сбор лута: новое оружие и зелья для лечения.
Смена оружия из инвентаря.
Скрытая пасхалка (секретная комната).

Другие аспекты:
Анимации с задержками для динамики (используя time.sleep).
Балансировка случайности (ограничены максимальные шансы на криты/комбо).
Поддержка групповых боёв.
